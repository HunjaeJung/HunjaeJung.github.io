Brief Introdunction of Neo4j

Written 150323
Tags: Neo4j

  1. Graph database 일단 그래프 데이터베이스는 말 그대로 그래프 자료구조를 기반으로 한 데이터 베이스입니다. 그래프는 Node와 Edge로 이루어졌으며, Node는 대상이나 개체를, Edge는 Node들 간의 관계를 나타냅니다. 그래프 데이터 베이스로 가장 잘 알려진 것은 neo4j인데, 그 사용법에 대해 차근차근 알아가봅시다.

  2. Getting started 일단 neo4j를 다운받습니다. 그 다음 압축을 풀고, terminal에서 /bin/neo4j start를 실행합니다. 그리고 http://localhost:7474로 접속해봅시다. 뭔가 멋진 것들이 뜨는데, 이곳에 있는 console에 쿼리를 날려 DB를 생성/수정/검색할 수 있습니다. 쿼리는 Cypher라는 neo4j에서 전용 쿼리 langauge를 사용합니다.

  3. Cypher language example

    아래는 Matrix 영화에 출연한 모든 배우들을 찾기 위한 쿼리입니다.

    ``` MATCH (movie:Movie)<-[role:ACTED_IN]-(actor:Person) WHERE movie.title="The Matrix" RETURN role.roles, actor.name

    Label=> ':Person', ':Movie' Relationship=> ':ACTED_IN' Property=> '.title, '.role', '.name' Identifier=> 'movie', 'role', 'actor' Value=> 'The Matrix' ```

    모든 데이터를 지우기 위해서는 아래와 같은 쿼리를 날립니다.

    MATCH (n) OPTIONAL MATCH (n)-[r]-() DELETE n,r
